锈湖鸡尾酒
——«锈湖：根源» 使用[锈湖]神秘的炼金术制造而成。
因为这地方太过邪门，我总是对这酒敬而远之。
火 + 土 = 岩浆空气 + 水 = 云
岩浆 + 空气 = 石头石头 + 火 = 金属云 + 土 = 植物
石头 + 空气 = 沙子金属 + 火 = 电植物 + 空气 = 树金属 + 石头 = 刀片
沙子 + 火 = 玻璃瓶电 + 玻璃瓶 = 灯泡树 + 刀片 = 木头刀片 + 木头 = 刀
植物 + 沙子 = 谷类玻璃瓶 + 水 = 一瓶水植物 + 灯泡 = 水果电 + 刀 = 榨汁机
谷类 + 一瓶水 = 伏特加水果 + 榨汁机 = 果汁
伏特加 + 果汁 = 〈锈湖鸡尾酒〉

谋杀烈酒
——«我的世界»荒川镇 〈谋杀烈酒〉，原名[大黑店黑桃J烈酒]，因其过于猛烈，喝下后仿佛受到持续伤害，像是要被谋杀一般，而有了“谋杀烈酒”这个别名。
把[烈焰粉]作为燃料放进[酿造台]，放入[水瓶]，加入[下界疣]酿造成[粗制的药水]，加入[蜘蛛眼]酿制成[剧毒药水]，不对，是〈谋杀烈酒〉。我不清楚有没有加入[荧石粉]增强效果。

四德酒
——«魔兽世界» 盲眼酒中仙[云壬]，由于双目失明其他感官变得更加敏锐，经过常年游历与试验终于研制出了能令他的四种感官分别感到满意的酒——〈四德酒〉。
传说〈四德酒〉已经失传，不过我在游历[潘达利亚大陆]时，在[不息山]最高的山顶找到了[云壬]的雕像，雕像旁的石板记录着〈四德酒〉的配方。
我在[四风谷]开辟了一片菜地，种植蔬菜，和当地的熊猫人以物易物，换取材料，终于酿造成功。
喝下后视觉会受到阻碍，但其他四感会变得异常敏感，据说可以免疫幻象。

酒神之酒
——«最终幻想14» 我接到委托去寻找三大美食和最上等的葡萄酒，酿酒师[夏马尼·洛马尼]提到〈酒神之酒〉，「那味道的力量堪比位于天界的十二神，将双目失明的我带到了一个色彩斑斓的世界中。」
但一通寻找无果，且酿造〈酒神之酒〉所需的[酒神葡萄]全部被毁。
正打算放弃，意外发现了[酒神葡萄]的叶子，顺着线索找到[酒神葡萄]的枝条，可以借此培育出[酒神葡萄]。
[夏马尼·洛马尼]将枝条交由酒厂老板[比格莱恩特]培育，[比格莱恩特]为表达感谢将私藏的一瓶〈酒神之酒〉送给了我。
交付委托后发现这场宴会是为我准备的，在宴会中吃光了三大美食，〈酒神之酒〉被我留下了。
我得到的这瓶酿制于〈星历〉1547年，被誉为“最高级古典葡萄酒”，并被冠以“梦幻”之名，是比稀有更稀有、比珍贵更珍贵的酒。

科尔戈的黄金
——«魔兽世界» 很久之前我曾听过一个传说，在[天涯海滩]有一处海盗的宝藏，当寻宝者历经千辛万苦凑齐藏宝图，踏遍险峰恶水，在死斗中斩杀其他寻宝的竞争者，终于打开宝箱——然而名为“黄金”的宝藏，只是一瓶酒而已。
我不知道那名寻宝者会是怎样的心情，也许是寻获宝藏的喜悦，也许是发财美梦破碎的失望？
但我很喜欢这个故事，在我看来这瓶酒具有令我着迷的传奇色彩。

我在写第一部«酒仙笔记»时有提到我的遗憾，有读者说他留有一瓶〈科尔戈的黄金〉，愿意与我分享。但我还是错过了。
但最终还是收集到了〈科尔戈的黄金〉，还有更稀有的〈有虫子的科尔戈黄金〉。
是«魔兽世界»里一个隐藏解谜任务，需要击杀很多海盗获得地图碎片，集齐上中下三张拼凑成藏宝图，跋山涉水找到藏宝地，还会窜出几个不怀好意的寻宝者，把他们都解决之后，打开的宝箱里有几率获得〈科尔戈的黄金〉或〈有虫子的科尔戈黄金〉或别的东西。
在我听说有这个任务的时候，这个任务已经被删掉了。
后来有了怀旧服，我去亲自体验了一番，终于得到了〈科尔戈的黄金〉和〈有虫子的科尔戈黄金〉。
在正式服有人送了我一瓶〈有虫子的科尔戈黄金〉，这时候这个酒已经绝版了十多年，应该是游戏里最稀有的酒了。
「如果在你手上收藏应该会更有意义。」他是这么说的。
非常感谢。

恐惧老酒
——«魔兽世界» 捡到[预言之卷]，拿去找[落日酒坊]帮忙破译。
当恐惧升腾，天空中战云密布，我们伟大的救世方舟，必会支离破碎。
造物者的撕裂之匕，融入林木与暗影，若日落之王的火焰之翼，得以永存不灭。
咒法妖精，千金之诺，熄灭远古的血与火。
理解这一神圣配方，照我所书执行，从远古之血中成果乍现，恐惧之战定将终结。
我和[丽亚·十歌]都觉得这是打造神器的配方，但[落日酒坊]的其他几个熊猫人坚持认为这是酿酒秘方，虽然起因有点离谱，但既然是酿酒，我肯定要帮忙了。
我们根据“造物者的撕裂之匕”这句，找来[巨型穆山兽牙]，[吴龙]：「我在想我们怎么把这些东西跟酒浆混合起来？不能直接碾碎它们，那样酒里会有很多沙子。或许我可以把它们泡一会儿。」
配方中提到“暗影”，[树脂大师吴暮]说他一直在考虑用一些这个地区残余的煞魔精华做为酿酒材料，我找来了一些，[树脂大师吴暮]：「非常好，添加这种材料可以让我们的酒更加香醇。最危险的材料通常都是最美味的。」
“融入林木”，我找来了些[芬芳的芯材]，[树脂大师吴暮]：「我们从不用木头酿酒，木头会让酒变得辛辣而寡淡，跟葡萄酒一样，我得对这些东西进行一下处理。」
“火焰之翼”，轻轻拍打精灵龙，从它们身上收集了许多[飞掠之尘]，将它们丢入锅内，[吴龙]评价：「这酒变得……很奇怪，而且很难闻。我开始怀疑[丽亚]或许是对的了。」
帮了[里基屯]的树精一些忙，作为回报，它们帮着做了些仪式。
最后是“远古之血”，[树脂大师吴暮]：「[丽亚]和我都认同一件事：‘远古之血’几乎可以肯定，就是当地的大树流出的树脂。我早该想到这一点的——我整天用那玩意儿酿酒。」
我找到了这里最古老的被称作“恐惧之心”的树，弄下一大块固化琥珀。
终于，我们的酒完成了。
[树脂大师吴暮]尝了一口，都惊呼道：「这太可怕了！」
他给酒取名“恐惧老酒”。
[丽亚·十歌]坚持认为「我想我们已经证明了这并不是一个啤酒配方。」
[吴龙]则依然兴奋：「耶！啤酒！！」
[树脂大师吴暮]：「谢谢你的帮助。我们也许没有酿出最好的啤酒，不过我们倒是经历了一场惊心动魄的冒险。」
没错，过程也是酒重要的组成部分。
从酒桶里可以接取〈一杯恐惧老酒〉，确实是可怕的酒——至少还能称作酒，那就没问题，嘿嘿！
之前[韩·风暴烈酒]被困在了琥珀里，我试着把〈一杯恐惧老酒〉倒在琥珀上，琥珀居然融化了。

小粉的烈酒
——«魔兽世界» 各种风味威士忌，采用了[艾泽拉斯]各地区独特的材料和工艺。
〈旧铁炉堡〉老式矮人鸡尾酒，经过铁匠和炉火炙烤的大麦酿造的麦芽酒。
〈布伦希尔达雾凇〉采用木柴火蒸煮的谷物酿造而成，还经过了独一无二的祛湿处理，在麦芽酒的甘冽和燥热口感中取得了完美平衡。
〈黄泽麦芽纯酿〉四风谷的精酿威士忌。酿造的秘诀是只使用来自黄泽瀑布下的水。
〈鹰巢8年陈酿，辛特兰麦芽酒〉劲爆的烈酒，使用只生长在辛特兰山巅的野麦酿造，故而具有野草的清香。
〈克莱因庄园12年陈酿〉按德鲁斯瓦传统制法酿造的威士忌，酒桶极其易燃，这一点可以善加利用。
〈霍斯瓦尔德25年陈酿，符文林地装桶〉符文林地的魔法特性使这种维库烈酒带有独特的回味。
〈阿拉索孤桶〉从古代的阿拉索帝国留存的唯一一桶酒里斟出来的，年代古老到无法考量。

神血蜜露
——«星穹铁道» 据传仅有十二瓶，是众泰坦各自神血浇灌的作物酿成，[法吉娜]埋藏于海床之中，以待千年后品尝，但随着黑潮的到来，这些酒没能被启封。
[阿格莱雅]：「尚显生机的时代里，但凡众城邦有幸起获之，无一例外将其视作宝藏，在邦交仪式中启封更象征着无上礼遇。」
  • ‘浪漫’的〈神血蜜露_浪漫〉：[阿格莱雅]送过我一瓶，金色的酒液。
  • ‘海洋’的〈神血蜜露_海洋〉：[海瑟音]请我喝过，紫色的酒液。
    说是其中最为甜美的一樽，与其他的[神血蜜露]不同，这樽会使人更加清醒，有着让人从海妖的迷魂歌谣中唤醒的功效，能驱散一切迷雾。
    味道有些复杂，[海瑟音]介绍「时而平静、时而猛烈，这便是‘海洋’的滋味」。

专属心酿
——«星穹铁道» [斯缇科西亚]城中有一个不大的圆形水池，紫色的液体，都是被踩烂的葡萄。
据[蜜酿主理人]介绍，曾经的[斯缇科西亚人]——乃至全[翁法罗斯]所饮的蜜酿，多数都是源自此处的果实，以及满溢之杯的神力。
这个水池正是蜜酿采榨区，是[法吉娜]的祭台，通过亲自挤压葡萄，人们来时的路便会立刻化作一瓶“专属心酿”，每瓶“专属心酿”的味道各不相同，「无论失却欢欣多久，都能以它平复心境，重拾笑靥……」
  • 海瑟音：〈深海〉，一瓶口感绝佳的蜜酿，堪比神庙里的精品…但似乎掺杂着硝烟、鲜血与泪水的味道。
  • 遐蝶：〈黯淡冥河〉，不含浓度的葡萄汁。似乎是因为一些用于发酵的生命已远渡冥界了……
  • 赛飞儿：〈粗制蜜酿！〉，粗制秘酿，就像是各个城邦市场售卖的，需要自己带个小碗去盛的那种便宜货一样——还怪好喝的。
  • 阿格莱雅：〈金色之露〉，烈到难以入口的纯酿，很难想象什么样的人才会喜欢这种口味，或是把这瓶送给别人。
  • 万敌：〈赤血蜜酿〉，虽然像是鲜血，但其实是100%纯石榴汁。悬锋人，实在奇妙。
  • 那刻夏：〈致大地兽〉，只有极少数人能品出，这樽蜜酿有多么价值不菲。铭为‘致大地兽’，光历4500年制。

荣誉骑士特调屋

  • 〈雪韵晨晖〉：传说是画家与酒保联合调配的杰作，以黎明时分的[龙脊雪山]为意象，配合酸奶油与可可的能量，能迅速唤醒饮用者身体中的活力…
