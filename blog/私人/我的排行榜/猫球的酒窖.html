〈锈湖鸡尾酒〉
 利用[锈湖]神秘的炼金术制造而成的鸡尾酒。
因为这地方太过邪门，我总是对这瓶酒敬而远之。
配方：
火 + 土 = 岩浆空气 + 水 = 云
岩浆 + 空气 = 石头石头 + 火 = 金属云 + 土 = 植物
石头 + 空气 = 沙子金属 + 火 = 电金属 + 石头 = 刀片植物 + 空气 = 树
沙子 + 火 = 玻璃瓶电 + 玻璃瓶 = 灯泡刀片 + 树 = 木头
植物 + 沙子 = 谷类玻璃瓶 + 水 = 一瓶水植物 + 灯泡 = 水果刀片 + 木头 = 刀
谷类 + 一瓶水 = 伏特加水果 + 榨汁机 = 果汁电 + 刀 = 榨汁机
伏特加 + 果汁 = 锈湖鸡尾酒
——«锈湖：根源»

〈谋杀烈酒〉——大黑店黑桃J烈酒
 原名“大黑店黑桃J烈酒”，因其过于猛烈，喝下后仿佛受到持续伤害，像是要被谋杀一般，而有了“谋杀烈酒”这个别名。
——«我的世界»荒川镇

〈四德酒〉
 盲眼酒中仙[云壬]，由于双目失明其他感官变得更加敏锐，经过常年游历与试验终于研制出了能令他的四种感官分别感到满意的酒——〈四德酒〉。
传说〈四德酒〉已经失传，不过我在游历[潘达利亚大陆]时，在[不息山]最高的山顶找到了[云壬]的雕像，雕像旁的石板记录着〈四德酒〉的配方。
我在[四风谷]开辟了一片菜地，种植蔬菜，和当地的熊猫人以物易物，换取材料，终于酿造成功。
喝下后视觉会受到阻碍，但其他四感会变得异常敏感，据说可以免疫幻象。
——«魔兽世界»

〈酒神之酒〉
 我接到委托去寻找三大美食和最上等的葡萄酒，酿酒师[夏马尼·洛马尼]提到〈酒神之酒〉，「那味道的力量堪比位于天界的十二神，将双目失明的我带到了一个色彩斑斓的世界中。」
但一通寻找无果，且酿造〈酒神之酒〉所需的[酒神葡萄]全部被毁。
正打算放弃，意外发现了[酒神葡萄]的叶子，顺着线索找到[酒神葡萄]的枝条，可以借此培育出[酒神葡萄]。
[夏马尼·洛马尼]将枝条交由酒厂老板[比格莱恩特]培育，[比格莱恩特]为表达感谢将私藏的一瓶〈酒神之酒〉送给了我。
交付委托后发现这场宴会是为我准备的，在宴会中吃光了三大美食，〈酒神之酒〉被我留下了。
我得到的这瓶酿制于〈星历〉1547年，被誉为“最高级古典葡萄酒”，并被冠以“梦幻”之名，是比稀有更稀有、比珍贵更珍贵的酒。
——«最终幻想14»

〈开拓者特调・“李猫球”〉
 我自己的特调，[惊梦酒吧]以我的名字保留在了酒单里。
[舒翁]：「你或许会离开，但你的故事永远不会被遗忘。」
配方：大型古典杯 + 少量冰块 + 星空香槟x1 + 激梦果酱x1 + 极致糖浆x1 + 冰点苏乐达x1 + 薄荷叶
——«星穹铁道»
