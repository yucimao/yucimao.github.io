挑水果

注意：制作前需要准备奶盖，水果茶需要准备腌制水果颗粒
茉莉花茶(绿研)泡法：10g茶 + 开水300g + 冰30g（冰块搅化后泡茶），6分钟后过滤 + 200g冰块搅化备用
茉莉毛尖茶汤：10g茶叶 + （开水400g+冰块30g）闷泡6分钟 + 冰块320g搅化备用
金观音/茉莉花/四季春茶汤：5g茶叶 + 开水200g闷泡10分钟 + 冰块180g搅化备用
水果颗粒：桃子/草莓/芒果/杨梅/葡萄 250g + 13g 冰糖浆腌制10分钟
椰奶冻：椰奶200g + 椰子水120g煮沸（不停搅拌） + 海藻晶冻粉26g边搅拌边煮1分钟后冷却凝固备用
葡萄果冻：28g水晶冻粉 + 30g白砂糖 + 140g热水 + 20g葡萄汁煮至沸腾后冷却凝固备用
冻冻：水晶冻粉50g + 开水300g + 黄金糖40g搅化煮沸后晾凉备用

品牌

桃子汁：达川、德馨
草莓汁：达川、佳美
杨梅汁：达川
葡萄汁：达川/佳美
橙汁：扬雅
