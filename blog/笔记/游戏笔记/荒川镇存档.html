由于是从服务器变为本地存档，一些插件和设置都没有了，进存档要尽快使用指令关闭火势蔓延，不然一些使用岩浆和火焰的建筑会被烧毁。
领地插件没有了，没法用领地传送，只能根据坐标传送，或者自己找。
后面我会放上我找到的一些建筑坐标，也欢迎大家补充。

指令

禁止火焰蔓延 /gamerule doFireTick false
白天/time set 0
夜晚/time set 14400
关闭时间流动/gamerule doDaylightCycle false
开启时间流动/gamerule doDaylightCycle true
