写酒仙笔记的习惯就是从«魔兽世界»开始的。
在别的游戏里，酒通常是一种功能道具，和血药、蓝瓶的作用类似，而在«魔兽世界»里不一样，这里的酒通常没有实际作用，反而会让玩家屏幕模糊、走路摇摇晃晃，这个设计对我来说太有趣了，我属于那种喜欢自己找乐子的休闲玩家，最喜欢这种没什么用的无聊小设计。醉酒效果记录
醉酒效果记录
  • 屏幕随醉酒程度越来越模糊。
  • 醉酒程度：有点醉醺醺的 > 酩酊大醉 > 烂醉如泥。
  • 走路、跑步、飞行、游泳都会随机改变方向，很难走直线。
  • 说话偶尔会在末尾自动加上“ ……嗝儿！”
  • 看到的敌人等级变低，最多5级。不过这只是“酒壮怂人胆”，没有实际的改变。
  • 饮酒过量会[醉酒呕吐]，昏迷几秒，并呕吐出绿色液体。
一些咖啡饮品可以用于醒酒。仅限游戏，现实中不要同时喝酒和咖啡，很伤身体。

相关传闻：
  • 喝酒会让敌人等级变低，消除等级差距导致的命中率降低。官方回应过，只是看起来等级低了，除此之外没有任何影响，“提高命中率”只是玩家的错觉。
  • 不同种族的酒量不一样，一个角色经常喝酒也会增加酒量。我测试过，没区别的。

而且这个游戏里的酒类道具超级多，设计的很用心，不同地区会出于当地的种族习惯、历史和地理等因素酿造出完全不同的酒，有些还会有一行趣味说明，通过酒可以从侧面了解这里的风土人情。
我开始在艾泽拉斯到处旅行，收集各式各样的酒，把我的银行仓库都占满了，我特意建立公会用于在公会银行存放它们。
可惜后来afk了几年，公会因长时间没上线被系统解散了，我的酒都没了。
最近（2024年）重新踏上艾泽拉斯的世界，开始重新记录，这次换了个新的风格，我会按照自己的喜好排序，不只是记录酒，还有相关的一些见闻。

熊猫人酒中仙
云壬 盲眼酒中仙[云壬]，由于双目失明其他感官变得更加敏锐，经过常年游历与试验终于研制出了能令他的四种感官分别感到满意的酒——〈四德酒〉。
我曾在考古时获得了[酒中仙云壬的拐杖]，解密获得的情报显示，由于[云壬]字迹太难辨认导致〈四德酒〉配方失传。配方就写在[不息山]最高的山顶上，[云壬]雕塑旁有个〈云壬的石板〉，天赋异禀的我成功还原了配方，还成功酿造了出来，就是价格不菲，所需材料当年大概要500金的成本才能做出一瓶。2025年是940金……好贵。
饮用获得buff〈四德酒_buff〉，会戴上一个红色的眼罩，眼界像是被白雾笼罩。
由于材料比较昂贵，加上介绍「提高你的警觉，使你的行动不受幻象的影响」，许多人都怀疑这酒有什么额外作用，但至今没有被发现。
配方：百年陈酱 + 米粉 + 黑胡椒 + 熊猫人蜜桃 + 红韭花x200
[熊猫人蜜桃]在烹饪材料商有卖，[百年陈酱][米粉][黑胡椒]在[半山集市]可以用[铁掌徽记]兑换，[红韭花]可以用[铁掌徽记]兑换或者自己种。1徽记才换25份，太贵了，如果拍卖行不贵的话还是直接买吧
坊间流传“四德酒”是翻译错误，真名应该是“四感酒（Four Senses Brew）”，意思是喝下酒之后你的视觉会受到阻碍，但其他四感会变得异常敏感。

尹心卧 伤心客[尹心卧]，他的[逸心酒]能让饮用者感受到温暖和幸福。据说他本人十分喜爱自己的酒，以至于每次卖出一桶酒都会伤心落泪。
很可惜已经失传，只能从考古中得到〈酒中仙尹心卧的空酒桶〉。
他的塑像位于[雷掌台]。

郭乾斗 阴阳拳[郭乾斗]，按照特定比例饮下他发明的[白灵酒]和[玄思酒]，酒会在胃里交融结合，令人飘飘欲仙，欣喜顿悟，但又不至于像重度酗酒者那样失去判断力和自制力的[黑白佳酿]。
从考古中可以得到〈酒中仙郭乾斗的鸳鸯杯〉，可惜的是这位酿酒仙师的秘方已经失传了，但这些精美的酒杯仍让人想起他那精湛的技艺。

罗纳·绿齿
「噢，一个客人，我这里的客人可不多。饿了吗？我做了很多好吃的东西，亲爱的。」
在[暗月岛]西部的森林里有一个女性亡灵，向迷路的游客兜售她在世界各地找来的美食，一口大锅不知道在煮着什么，手里的剔骨刀还流淌着鲜血。
她以前会卖〈暴风惊爽酒〉，但从〈第7时代〉开始，她的商品就少了许多，这种酒也没了。
罗纳·绿齿的真相
罗纳·绿齿的真相 原型可能是英国传说中的水鬼“绿牙珍妮”。
她售卖的食物都有点不对劲，比如来自[提瑞斯法林地]的〈被遗忘者鹅肝酱〉，可那里没有鹅；来自[诺莫瑞根]的〈诺莫乐鸡块〉，可那里的鸡都是机械；来自[杜隆塔尔]的〈绿火腿配蛋〉，那里的兽人是绿色的……
也就是说，她卖的东西很有可能就是用当地种族做成的，那么〈暴风惊爽酒〉就是[暴风城]的种族，也就是用人类酿成的酒了。

黑铁酒吧
 我有一个〈烈酒的遥控器〉，可以召唤出一台[个人钻探机]，用于传送到[黑铁酒吧]。（遥控器在〈美酒节〉获得）
之前有过一次修改，让它变得很难用，我做了相关攻略，好消息是现在已经改回来了。
在这里喝酒的大多都是些喝醉的奴隶主，快乐的跳着舞。虽然对我不太友好，但也是[黑石深渊]里难得的休战区域。
这里有性感魅魔招待[娜玛拉小姐]，有独特的[黑铁啤酒]，还有艾泽拉斯最好的乐队[牛头人酋长乐队]每晚8点都会在这里演出，还可以找工作人员通融一下，让他安排一场“排练”，这简直是在为我单独演奏，实在是太酷了！（对着乐队成员使用动作“/欢呼”还能获得buff[激动]，所有属性提高10%，持续5分钟。）
酒吧的老板是一个侏儒，名为[普拉格]，据说他以前是个“与魔鬼打交道的术士”，但现在以自己酒吧老板的身份引以为傲，确保让顾客的酒杯里永远美酒满溢。
在这里可以买到〈黑铁酒杯〉〈钻探工的黑石啤酒〉〈萨弗隆烈酒〉，其中〈萨弗隆烈酒〉饮用会受到持续3秒的火焰伤害，这个效果会被视为“持续伤害法术”，以前有人会在打架前喝几口用来触发需要“持续伤害法术”的装备效果。“钻探工的黑石啤酒”是翻译错误，原文“Plugger's Blackrock Ale”意思是“普拉格的黑石啤酒”。
〈美酒节〉期间，喝多之后能在这里看到一只[粉色雷象]，清醒后就消失不见了。
在[苏拉玛]的一位“稀有进口物品商”那里买到过〈腌制卤蛋〉，据说是「黑铁酒吧最受欢迎的菜」，但我不太信得过这个商人，也许是假的。

始祖龟的水下酒吧
 有些地区的水下可能会隐藏着一间始祖龟的酒吧，需要潜水下去才能找到。
不是水下的建筑，它就在水里，在珊瑚、贝壳、海星和水草之间，酒客们也泡在水里，身边还会有小鱼游过，连鱼都可能是醉醺醺的，很神奇的体验。
阳光被海水遮挡，让酒吧有些昏暗，但很热闹，有水母充当迪斯科灯球，海龟驮着酒杯充当服务员，还会有海妖作为驻场歌手。
酒吧的老板都被称作“轶闻大师”，顾客有海地精、海巨人、锦鱼人、娜迦等水下种族，也会有陆地生物，在酒吧附近会获得某种祝福，不需要呼吸氧气。
他们的酒都是我闻所未闻、见所未见的，而且几乎都是烈性酒，不只是仿佛能杀人一般的度数，这些酒看起来似乎真的能要我的命。如果喝腻了常规的酒（并且有足够的勇气），可以来试试。

〈樱桃酒〉 「不论是要到古老遗迹或森林野地探险，对冒险者来说最重要的就是健康。这杯滋味香甜、颜色深沉、酒精浓度高而且有点呛口的饮料，是征服敌人的美味手段。」
技能等级：初级 / 准备时间：5分钟 / 分量：2人份 / 适合搭配：哥布林奶油酥饼
准备材料：姜汁啤酒6盎司 + 朗姆酒4又1/2盎司 + 酸樱桃果汁3盎司 + 青柠汁1又1/2盎司 + 青柠片与樱桃（用来装饰）

〈冬天爷爷的麦酒〉 「要是没有喝上一大杯这种令人暖上心头的美味饮品，哪能称得上冬幕节！」
技能等级：大师级 / 准备时间：10分钟 / 烹饪时间：30分钟 / 分量：约8人份 / 适合搭配：酒糟发酵甜甜圈、烤猪肉
准备材料：小苹果4颗 + 红糖1杯 + 肉桂粉1大勺 + 柳橙1颗 + 丁香粒1大勺 + 艾尔啤酒3瓶（1瓶约340cc） + 苹果酒1至3品脱 + 白兰地1/2杯 + 枫糖浆1/2杯 + 节庆香料1大勺 + 肉桂棒2根 + 蛋6颗，蛋白蛋黄分开
  1. 烤箱预热至华氏350度（约摄氏177度）。准备一个烤盘，并在烤盘上铺满烘焙纸。
  2. 用一个水果挖球器或锯齿水果勺，在不切开苹果的情况下挖掉果核，制造出一个小凹洞。糖、肉桂粉混合后平均分到全数挖好的苹果内。在柳橙上截出适当的洞（视需要可使用削皮刀或水果刀），塞入丁香粒。把准备好的苹果与柳橙放到(1)的烤盘上，放入烤箱中烤约20至30分钟，或直到苹果变软但尚未化开之前出炉。
  3. 趁着水果在烤箱里的期间，在一个大双耳锅混合除了蛋以外的其他材料，并以中火加热。
  4. 在一个中碗里搅打蛋黄约1分钟，直到蛋黄变成淡黄色。另取一个干净的碗，打发蛋白直到呈现硬性发泡的状态。轻轻地把蛋黄搅拌进打发的蛋白里，接着倒进1杯分量的(3)，期间持续轻轻搅拌，让蛋液稍微升温。把剩下的(3)倒进另一个鸡尾酒碗，加入前述混合的蛋液。放进苹果与柳橙，让它们漂浮其上。上桌时请装盛进至耐热马克杯里。
备注：每一颗苹果的受热情况不一定会一样，但就算苹果被烤到化开了，还是可以把它加进鸡尾酒碗里，这其实就像把苹果换成苹果酱而已，无伤大雅。

〈热苹果酒〉 「只需啜饮一口热苹果酒，热气腾腾又飘着令人温暖的淡淡的香料味，马上就有驱寒的效果。」
技能等级：高级 / 准备时间：15分钟 / 分量：4至6人份 / 适合搭配：慢烤火鸡
准备材料：苹果酒1/2加仑，有无气泡均可 + 节庆香料1小勺 + 白兰地1至2杯 + 红糖2大勺 + 肉桂棒（用来装饰） + 柳橙皮（用来装饰）
在一个中双耳锅里混合装饰用途之外的所有材料并以中火加热。稍微沸腾后继续熬煮10分钟，接着关火并倒入耐热马克杯。稍微扭转柳橙皮，挤出些许油脂。在每个马克杯里放1根肉桂棒、1条柳橙皮即可享用。
备注：最适合调制这种饮品的是苹果白兰地，不过苏格兰威士忌、雪莉酒、朗姆酒也很适合喔。

技能等级：高级 / 准备时间：10分钟 / 浸泡时间：30分钟 / 分量：约10人份 / 适合搭配：布利起司与水果果酱
准备材料：白砂糖1/3杯 + 白兰地1杯 + 石榴果汁1杯 + 白橙皮酒1/2杯 + 苹果2个，去皮、去核、切小丁 + 石榴1杯 + 柳橙2个，切片再切成小扇形 + 大颗青苹果1个，去核、切丁 + 依个人喜好添加诸如热带水果、草莓、覆盆莓等其他水果（也可不加） + 有果味的红酒2瓶（1瓶约750cc）

〈甜果酒〉 「如果您是愿意为了一杯美酒，就算赴汤蹈火也在所不惜的人，这份食谱绝对会让您的英勇冒险值回票价。」

〈月色酒〉 「月光林地的德鲁伊非常懂得享乐，这种甜甜的、非常容易调配的饮料是庆祝新年节庆时不可或缺的饮品。」

备注：用紫外线灯照射更能展现出这杯调酒的特殊风情。

〈熊猫人青梅酒〉 「水果！还有气泡！这玩意要是喝太多，小心就连醉鬼贾古都会比您清醒。」

准备材料：清酒1又1/2盎司 + 柳橙香甜酒1又1/2盎司 + 梅子果酱1大勺 + 冰块1/2杯 + 冰镇气泡葡萄酒或香槟（用来调味）
雪克杯放入清酒、柳橙香甜酒、果酱与冰块。盖上盖子轻快摇晃数次，平均过滤进两个薄型玻璃酒杯里。最后倒进一些许气泡葡萄酒即可享用！

〈南岛冰茶〉 「对大灾变感到哀伤吗？喝一杯南岛冰茶您就不会有这种感觉了。加了凤梨让传统配方迸出新滋味。」

在一个雪克杯里加进所有酒类与酸甜汁，盖上盖子用力摇晃几秒。倒进放好冰块的细长玻璃杯，最后倒进些许凤梨汽水。

〈冬幕节蛋奶酒〉 「这种加入烈酒的调酒，是冬天爷爷最爱用来搭配姜饼的饮料。」
技能等级：大师级 / 准备时间：10至15分钟 / 分量：6至8人份 / 适合搭配：姜饼
准备材料：全脂鲜乳3杯 + 白砂糖1杯 + 蛋6颗 + 鲜奶油2杯 + 香草精1又1/2小勺 + 朗姆酒1/4杯 + 白兰地1/4杯 + 现磨肉豆蔻
在一个大单柄锅里加进鲜乳与糖，以中火加热。期间快速将蛋打散并倒入锅里。继续加热10至15分钟，期间一样要不断搅拌，直到整体明显变得浓稠。关火，过滤倒进另一个干净的水壶或碗。加进鲜奶油、香草精、朗姆酒与白兰地。放进冰箱准备随时享用，但要记得最多只能冷藏数日。在饮用前可在杯子里淋上些许肉豆蔻粉。

«艾泽拉斯的新口味»
 我的烹饪学徒〈诺米〉已经成长为了著名的熊猫人大厨，他在[艾泽拉斯]各地旅行，收集整理出了这本食谱。
书里介绍〈诺米〉：
著名的熊猫人厨师〈诺米〉在半山新鲜诺农场开始学习烹饪，与伟大的厨师和旅行冒险家一起学习。小时候在潘达利亚，他展现了创造力，能够用各种新的不寻常的方式来烧焦食物。离开家后，〈诺米〉周游艾泽拉斯，结识新朋友，学习新食谱，探索世界。作为一名经验丰富的厨师，〈诺米〉喜欢与任何愿意学习的人分享他的食谱。无论一道菜有多复杂，或者一种食材有多难运用，〈诺米〉都能确保做好他的食物。

[沙滩机甲] ——塔纳利斯 「不久前，我在[塔纳利斯]结识了一位名叫[菲兹格瑞博]的旅店老板，他在忙碌了天之后为我调了一杯这种酒。味道太棒了，你能想象自己躺在某个宁静的阳光沙滩上，而不是被地精城市里无所不在的机械噪声的喧嚣声包围。」
技能等级：旅者 / 准备时间：5分钟 / 量：1份 / 适合搭配：各种咸味零食
准备材料：龙舌兰酒30ml + 桃味利口酒30ml + 橙汁60ml + 石榴汁60ml + 青柠片，装饰用
将龙舌兰酒、桃味利口酒和橙汁一起倒入鸡尾酒调酒器，用力摇晃几下。将石榴汁倒入一个放有冰块的高脚杯，然后将橙汁混合物倒在上面。用新鲜的青柠片做装饰。

〈蜜糖火酒〉 ——厄运之槌 「即使是最强大、最凶狠的战士，偶尔也会要在睡前喝上两口。[践踏者克雷格]从[厄运之槌]的竞技场退役后，我从他那里得到了这份食谱。跟[践踏者]一样，这种烈性利口酒有一种很强的冲击力，但只喝一小杯的话，它算是很棒的餐后酒。」
技能等级：旅者 / 准备时间：5分钟 / 静置时间：48小时 / 量：1小瓶，若干份 / 适合搭配：布里维普的冰淇淋
准备材料：去皮切碎的生姜30g + 小肉桂棒1根 + 1颗小橙子的皮 + 丁香粒3颗 + 香草提取物少量 + 糖蜜120ml + 水120ml + 黑红糖25g + 白兰地240ml + 朗姆酒120ml
  1. 在一个小深煮锅里，混合除朗姆酒和白兰地之外的所有食材，然后用中低火煮至快开的状态。再煮5分钟左右，将所有东西倒入一个宽口瓶里。加入白兰地和朗姆酒，轻轻盖上瓶盖，静置48小时。通过滤网倒入一个干净的瓶子里，然后在阴凉处存放。
  2. 这种酒刚开始会比较辣口，不过，再等上个把星期，口感就会变得醇厚了。

〈符文林地白兰地〉 ——英灵殿 「作为[风暴峡湾]的一种特产，人们喝这种蒸馏酒是为了提升精力，原因也不难猜到。只须抿一小口——特别是在一顿大餐之后，就能让你精神焕发，可以应付接下来的任何情况。据说在[英灵殿]的宴会桌上，有一杯永不干涸的[白兰地]。」
技能等级：学徒 / 准备时间：5分钟 / 浸泡时间：3~4天 / 量：1瓶，约1L / 适合搭配：饱餐一顿后放松一下，森林远足
准备材料：香菜籽5g + 小豆蔻2g + 葛缕子2g + 黑胡椒粒6颗 + 丁香粒2颗 + 新鲜莳萝1小枝 + 伏特加1L
  1. 用杵臼轻轻磨碎各种香料，然后与莳萝和伏特加一起倒入一个干净的大瓶子里。浸泡几天，偶尔摇晃一下，然后通过滤网倒入一个干净的瓶子中。
  2. 想要口味更加浓烈，你可以让浸泡时间延长至两个星期。一些冒险家喜欢放入蜂蜜或糖来稍微增加甜味，所以你也可以根据自己的喜好来调味。

〈热砂货车炸弹〉 ——艾尔文森林 「我一直不明白为什么这种饮料没能流行起来。我是说，我打倒了数量惊人的[鱼人潜伏者]，才找到一个身上带着这饮料的。也许我得跟[热砂保护协会]的人好好谈谈这事……」
技能等级：旅者 / 准备时间：5分钟 / 量：1人份 / 适合搭配：海盐焦糖
准备材料：烈性苹果酒180ml + 酸苹果蒸馏酒30ml + 肉桂味威士忌30ml
  1. 将烈性苹果酒倒入大玻璃杯。
  2. 将酸苹果蒸馏酒和肉桂味威士忌倒入小酒杯里混合。
  3. 准备喝时，要么将小酒杯直接扔到大玻璃杯里然后一饮而尽，要么将小酒杯里的混合酒倒入烈性苹果酒中，然后慢慢品尝。

〈魔吉托〉 ——达萨罗 「浓浓的柑橘味！强劲的朗姆酒！咕噜噜翻腾的泡泡！所有这些，再加上其他一些原料，完美地结合在一起，调制出了极其清爽又艳丽夺目的鸡尾酒，再热的夏天它都可以使你安然度过。」
技能等级：学徒 / 准备时间：5分钟 / 量：1人份 / 适合搭配：洛阿神灵面包，冰镇水果沙拉
准备材料：新鲜薄荷叶1把 + 朗姆酒50ml + 青柠汁一点点 + 龙舌兰糖浆15~30ml，调味用 + 冰块 + 普洛塞克或其他起泡酒30~50ml，或者根据口味加更多 + 新鲜青柠片，装饰用
把新鲜薄荷叶放在平底玻璃杯底部，倒入朗姆酒、青柠汁和龙舌兰糖浆，然后将新鲜薄荷叶捣碎。你也可以使用手持搅拌机来搅拌这一杯混合物，但你可能需要将食材的量加倍，这样才有足够的液体用于搅拌。加入冰块，再倒入起泡酒加满杯子。用新鲜青柠片装饰，即可享用。
〈诺米〉备注：制作鸡尾酒需要捣碎？所谓“捣碎”就是将食材往杯壁上挤压，使其破碎并释放出它的味道。这样做有助于食材味道更好地与酒结合。

〈石南马天尼〉 ——纳兹米尔 「来自祖达萨的另一种美酒。赞达拉的巨魔在一天辛苦的钓鱼、划船或寻宝等活动后最喜欢喝的一些烈性酒，这里都有。此外，这款鸡尾酒还加入了浆果果酱和柑橘类果汁，口感更清淡和顺滑。」
技能等级：旅者 / 准备时间：5分钟 / 量：1人份 / 适合搭配：奶油酥饼
准备材料：琴酒60ml + 干味美思酒30ml + 糖粉3g + 柠檬汁5ml + 玫瑰水少量（可选） + 覆盆子酱7g + 冰块 + 新鲜浆果，装饰用
在调酒器中混合所有的食材，用力摇晃几秒钟。通过滤网倒入鸡尾酒杯中，然后用新鲜浆果装饰。
〈诺米〉备注：如果你想让它提高一个档次，可以往调酒器里加入7g覆盆子酱。你将会品尝到经典的刺激味道，还能获得更多的维生素！

〈疯酒师的早餐〉 ——四风谷 「这是来自大师[博布·铁掌]的食谱，老实说，我在旅途中从未见过类似的东西。他发誓每天早上都要喝一杯这种酒，还要进行严格的冥想，为的是保持健康长寿。」
技能等级：高级 / 准备时间：10分钟 / 量：1人份 / 适合搭配：更多培根
准备材料：鲜榨青柠汁15ml + 合适辣度的绿色沙拉60g + 青苹果半颗，去核然后切碎 + 小菠菜15g + 水120ml + 原味酸奶60ml + 辛辣香料15g，粘杯沿 + 新鲜苹果，装饰用 + 熟培根，装饰用 + 圣女果，装饰用 + 新鲜欧芹枝，装饰用
将所有食材放入玻璃罐或者碗中，使用搅拌器搅拌均匀。通过滤网倒入一个玻璃杯，饮用前先冷藏并配上装饰物。
〈诺米〉备注：我个人很喜欢这种饮料的简单版本，但大师[博布·铁掌]喜欢打一个生鸡蛋进去。真是一口闷的早餐啊！甚至有人会往里面加半两或更多的烈酒，做成一顿可以喝的大餐！

〈影莓香蒂酒〉 ——雷文德斯 「忙碌了一天之后，这种漂亮的鸡尾酒很容易就能一口闷了。它充满气泡、果香四溢，一定能让你精神为之一振。浓郁的柑橘味和多汁的浆果，当你在忙碌的日子里停下来歇歇时，这种清新口感非常棒。」
技能等级：学徒 / 准备时间：5分钟 / 量：1人份 / 适合搭配：炎热的夏日，鸦莓小圆饼
准备材料：淡啤酒360ml，最好是拉格啤酒，冰镇 + 接骨木莓或黑加仑浓缩糖浆40g，比如利宾纳 + 柠檬汁少许 + 柠檬角，装饰用
将淡啤酒顺着杯壁倒入杯中，不要让它起太多的泡。将接骨木莓浓缩糖浆直接倒入淡啤酒里，然后加入柠檬汁。趁冷喝。

〈熔岩鸡尾酒〉 ——大漩涡 「我的好朋友[格莱维]表示，如果你发现自己被熔岩包围，最好的办法就是喝一杯冷饮来抵御高温！他发誓这种清爽的饮料是他原创的。他得明白，这是他店里最受欢迎的商品之一！」
技能等级：高级 / 准备时间：10分钟 / 量：2杯 / 适合搭配：辛辣食物，熔岩烤香肠
准备材料：冰冻菠萝块250g + 橙汁240ml + 冰冻草莓250g + 椰汁240ml + 红石榴糖浆（可选）15~30ml + 新鲜菠萝，装饰用
  1. 混合冰冻菠萝块和橙汁，然后分成两杯。
  2. 混合冰冻草莓和椰汁，有红石榴糖浆的话也加进去。把这个混合液轻轻地倒在菠萝橙汁上。用新鲜菠萝装饰，即可饮用。
〈诺米〉备注：想喝这种酒的微醺版本，可以用朗姆酒代替一半的橙汁。

〈茴香烧酒〉 ——北风苔原 「乍一看，用数量稀少的库尔提拉斯酒来制作茴香烧酒似乎并不划算，但当你喝过一次之后就不会这么想了。在严寒的北风苔原，没有什么比烧酒更能温暖你内心的了。」
技能等级：学徒 / 准备时间：5分钟 / 静置时间：10天 / 量：约2斤 / 适合搭配：意式杏仁脆饼
准备材料：葛缕子5g + 丁香粒2~3颗 + 肉桂棒1根 + 柠檬皮丝 + 白兰地1L + 蜂蜜80g，或者根据口味加更多
