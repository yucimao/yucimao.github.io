[鲜虾鸡尾酒]其实不是酒，是一种开胃菜。在官方短片«5 Days Of Christmas»里，[鲜虾鸡尾酒]的杯子里还泡着一颗眼球。
〈鹿先生〉想喝〈血腥玛丽_1〉。
找到了配方： [血] + 〈伏特加〉 + [辣椒酱] + [秘密成分？] = 〈血腥玛丽_1〉
用刀划开〈鹿先生〉的鹿角，获得[血]。
在沙发后面找到〈伏特加〉。
从挂在墙上的鹿头里取出[辣椒酱]。
剖开蝴蝶，取出[种子]，种进土里，从长成的植物里采集[有毒的种子]，放进搅碎机里，获得[毒药粉]，用虹吸壶煮成[毒药]。
将这些成分倒进玻璃杯里，就成了〈血腥玛丽_1〉。

〈h3|«剧院»〉 〈伏特加〉 x2 + [橙汁] x2 = 〈螺丝起子鸡尾酒〉
〈鲍勃〉喝完嘴里吐出螺丝刀。
因一部侦探剧中角色的台词「喝[螺丝起子]现在还太早了点儿」而流行起来，从此这种酒成了侦探剧的标配。

〈杜松子酒〉 x2 + [汤尼水] x2 = 〈琴汤尼〉«生日»里出现过这个配方，成就里这个酒叫[锈湖琴汤尼]，在«悖论»的电话簿里有它的广告。

[橙汁] x2 + 〈加里安奴酒〉 + 〈伏特加〉 = 〈哈维的伏特加橙汁鸡尾酒〉
〈鲍勃〉喝完疯了一样用头撞桌子，直到磕出血。恢复清醒不记得刚才做了什么。
起源是一个名为[哈维]的人，为了庆祝自己得了游泳冠军，在酒吧喝加[橙汁]的[伏特加]，他喜欢在其中加[加里安奴酒]，连喝了十杯，走出酒吧不小心撞到了墙上，这种酒因此得名。

〈伏特加〉 + [塔巴斯哥辣酱] + [血] x2 = 〈血腥玛丽_2〉
〈鲍勃〉喝完掏出枪对着太阳穴开了一枪。
在«白门»里提到，这是〈鲍勃〉最喜欢的饮料。
