酒吧

伏特加 x2 + 橙汁 x2 =
螺丝起子鸡尾酒

喝完吐出螺丝刀。
因一部侦探剧中角色的台词“喝螺丝起子现在还太早了点儿”而流行起来，从此这种酒成了侦探剧的标配。

杜松子酒 x2 + 汤尼水 x2 =
琴汤尼


橙汁 x2 + 加丽安奴酒 + 伏特加 =
哈维的伏特加橙汁鸡尾酒

喝完疯了一样用头撞桌子，直到磕出血，恢复清醒不记得刚才做了什么。
原型是哈维撞墙鸡尾酒，起源是一个名为哈维的人，为了庆祝自己得了游泳冠军，在酒吧喝加橙汁的伏特加，他喜欢在其中加加丽安奴，连喝了十杯，走出酒吧不小心撞到了墙上，这种酒因此得名。

伏特加 + 塔巴斯哥辣酱 + 血 x2 =
血腥玛丽

喝完掏出枪对着太阳穴开了一枪。

石头 + 空气 = 沙子沙子 + 贝壳 = 粉笔粉笔 + 卵 = 蛋
沙子 + 火 = 玻璃瓶玻璃瓶 + 蛋 = 长生不老药
